# Configure application

1. Clone repository

2. Install dependencies
```
yarn install
```

3. Install Nest CLI
```
npm i -g @nestjs/cli
```

4. Start up database
```
docker-compose up -d
```

#
# Start application

1. Start
```
yarn start
```

2. Start mode developer
```
yarn start:dev
```

3. Rebuild Database with seed
```
http://localhost:3000/api/v2/seed
```
